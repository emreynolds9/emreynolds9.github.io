Hello! I am a data analyst in the greater Washington, DC area.

In 2019, I completed an intensive 24-week Data Analytics Boot Camp at George Washington University where I developed my data analysis, cleaning, and visualization skills, along with application development. You can see the complete syllabus for the program here.

This website serves as a portfolio of my favorite projects from the course. I built the site from scratch as an exercise in responsive website development.

Machine Learning: Housing Affordability and Technology

This project consisted of a machine learning model that estimated the influence of the technology sector on the housing market of a county. The data was trained on over 1,000 counties in the United States.

Matplotlib: Weather Patterns & Latitude

This visualization utilizes the OpenWeather API to build a data set of weather conditions at coordinates around the world. I used Python and Matplotlib to compare weather patterns such as precipitation and humidity to the latitude of the observed location.

  • Website
  • Github Repository

Leaflet: Mapping Recent Earthquakes

This visualization utilizes the United States Geographical Survey API to map global earthquakes in the last seven days. I used JavaScript and Leaflet to create an interactive map that changes in real time.
