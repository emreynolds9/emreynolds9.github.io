This project was a visualization exercise in which I used a weather API to compare weather patterns such as precipitation and humidity to the latitude of the observed location.

Mapping Recent Earthquakes

This project was a visualization exercise in which I used the United States Geographical Survey API to map earthquakes in the last seven days. The visualization is live and changes in real time.
